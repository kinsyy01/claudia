Equal Count (Quantiles) classification

The Equal Count (Quantiles) method classifies data into a certain number of categories with an equal number of units in each category. This classification adequately displays data that varies linearly.

Equal Interval classification

Equal Interval classification sets the value ranges in each category equal in size. The entire range of data values (max - min) is divided equally into however many categories have been chosen.

Natural Breaks classification

Natural Breaks designates the division of values to the program and classes tend to be more similar in value.
