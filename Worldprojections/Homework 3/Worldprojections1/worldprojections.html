In this project I learned how to display the world in different projections.

Describe in your own words how you displayed the map in different projections using QGIS.

Using the data Professor Arce-Nazario gave us, I utilized Visual Studio Code and GitHub to display the various projections found below. I added a vector layer to create the base map and used Indicatrix Mapper to display the dots. Afterwards, I would click on the projection tab to search for the projection I wanted.

WGS84 Projection

Distorts size closer to the poles. For example, Greenland looks similar to the size of the United States, which is not true in reality.

Aitoff Projection

Distorts size of continents near the edges of the projection and distorts relative location.

WGS 84 / Pseudo-Mercator

Antarctica appears huge. Greenland appears to be similar to the size of Africa. Size is distorted near top and bottom of map.

Sphere Winkel I

Sizes of countries seem projected realistically, or truer to actual size.

World Cylindrical Equal Area

Area has been lost near top and bottom of map, almost flattened. Map is shaped like a rectangle which distorts relative location but preserves size.

World Equidistant Conic

Map is centered around a central axis and is curved around it to appear more conical in nature. Data is not projected beyond an angle near the top of the map, which wastes space.

North Pole Azimuthal Equidistant

Very similar to previous map, however Antarctica is preserved and projected all around the outermost layer. The Arctic is placed neatly in the middle.
